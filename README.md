# EduSmart School - Plataforma Educativa

## 📚 Descripción
EduSmart School es una plataforma educativa integral diseñada para facilitar la gestión académica en colegios. Permite la interacción entre administradores, profesores y alumnos en un entorno digital seguro y eficiente.

## 🚀 Características
- ✅ Sistema de autenticación con Firebase
- ✅ Paneles personalizados por rol (Admin, Profesor, Alumno)
- ✅ Gestión de recursos educativos (PDF, imágenes, videos)
- ✅ Seguimiento de progreso académico
- ✅ Datos de ejemplo para demostración

## 🛠️ Tecnologías Utilizadas
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Firebase (Auth, Firestore, Storage, Hosting)
- Font Awesome para iconos

## 📋 Requisitos Previos
- Node.js y npm instalados
- Cuenta de Firebase
- Firebase CLI instalado (`npm install -g firebase-tools`)
